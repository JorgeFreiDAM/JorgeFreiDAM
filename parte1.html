HTML

  1. ¿Qué es HTML?

    • Lenguaje de marcado estándar.
    • Consiste de una serie de elementos.
    • Le dicen al navegador cómo mostrar los elementos.

  2. Elementos y etiquetas

    3. HTML se compone de elementos que forman la estructura de una página web.
    Los elementos estan formados por dos etiquetas: la etiqueta de apertura y la de cierre.
    El nombre del elemento se incluye entre los signos "<"   y ">", en la etiqueta de cierre el nombre del elemento se precede por "/".
    <p>Esto es un nuevo párrafo.</p>

  3. Estructura básica

    4. <!doctype html>
     <html lang="es">
      <head>
       <title>Título de la página web</title>
      </head>
      <body>
       Texto del documento
      </body>
     </html>

    El elemento <head> contiene metadatos (datos sobre el propio documento).
    Suelen definir el título, la codificación de carácteres, estilos o scripts.

    El elemento <body> representa el contenido de un documento HTML. Solo puede haber uno en un documento.

    El elemento <title> es requerido, define el título del documento. Se muestra en la pestaña del navegador.

  4. Metadatos

      5. Se incluyen dentro del elemento <head>.
    • <base>
      Especifica una URL predeterminada para todos los enlaces relativos del documento (se "añade" al principio del resto de hrefs del documento). <base href="index.html">
    • <link>
      Sirve para enlazar JavaScript y CSS externos con el documento actual. <link rel="stylesheet" href="style.css">
    • <style>
      Aplica CSS desde el propio documento HTML. <style>..CSS..</style>

  5. Encabezados

    6. Representan títulos o subtítulos, se definen con las etiquetas <h1>, <h2>.. <h6>
    <h1> define el título más importante y <h6> el menos importante.  <body>
      <h1>Encabezado1</h1>
      <h2>Encabezado2</h2>
      <h3>Encabezado3</h3>
      <h4>Encabezado4</h4>
      <h5>Encabezado5</h5>
      <h6>Encabezado6</h6>
     </body>

  6. Secciones

    7. Agrupan el texto en secciones temáticas.
    • <nav> Define una sección con enlaces de navegación.
    • <article> Contenido autónomo que puede existir con independencia del resto de contenido.
    • <aside> Contenido poco relacionado con el resto del contenido de la página.
    • <header> Cabezera de la página o sección.
    • <footer> Pie de página o sección.
    • <adress> Información de contacto.
    • <main> Contenido principal o importante del documento, solo puede haber uno.
    • <section> Sección genérica de un documento.

  7. Agrupación de contenido

    8. Ayudan a dar formato al texto según su contenido.
    • <p> Crea párrafos nuevos o define una sección como párrafo.
    • <hr> Cambio temático entre párrafos.
    • <div> Contenedor genérico sin ningun significado.
      • Listas
      • <ul> Lista desordenada.
      • <ol> Lista ordenada.
      • <li> Elemento de una lista.
     <body>
      <p>Esto es un párrafo</p>
      <hr>Cambio temático</hr>
      <ol>
       <li>primer elemento de la lista.</li>
       <h5>segundo elemento.</h5>
      </ol>
     </body>
